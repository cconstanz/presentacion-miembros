Arquitectura de Microservicios

Proyecto ms-miembros

El microservicio núcleo de la plataforma FitLife que centraliza la gestión de usuarios y garantiza la seguridad mediante autenticación JWT.

Ver Documentación / Swagger API

La Propuesta de Valor

❌ El Problema

Las plataformas de gestión monolíticas tradicionales sufren problemas de rendimiento cuando la base de datos de usuarios crece. Además, carecen de métodos de autenticación robustos, exponiendo la información personal de los clientes y generando cuellos de botella al integrarse con otros módulos.

💡 La Solución

ms-miembros desacopla la lógica de negocio usando Spring Boot y PostgreSQL. Implementa seguridad Stateless con tokens JWT, lo que permite escalabilidad independiente, alta disponibilidad en la nube (Render) y una comunicación segura con los servicios de gamificación y notificaciones.
